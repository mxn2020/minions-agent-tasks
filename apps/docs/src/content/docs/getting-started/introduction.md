# Minions Agent-tasks

gh agent-task prompts, execution tracking, PR references, and review status

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-agent-tasks/sdk`) — Core library for Node.js
- **Python SDK** (`minions-agent-tasks`) — Python bindings
- **CLI Tool** (`@minions-agent-tasks/cli`) — Command-line interface
- **Web App** — Interactive playground at [agent-tasks.minions.wtf](https://agent-tasks.minions.wtf)
