---
title: Installation
description: Install Minions Agent-tasks packages
---

## TypeScript / Node.js

```bash
npm install @minions-agent-tasks/sdk minions-sdk
```

## Python

```bash
pip install minions-agent-tasks
```

## CLI

```bash
npm install -g @minions-agent-tasks/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
