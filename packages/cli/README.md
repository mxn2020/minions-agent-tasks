# @minions-agent-tasks/cli

CLI for Minions Agent-tasks — gh agent-task prompts, execution tracking, PR references, and review status

## Installation

```bash
npm install -g @minions-agent-tasks/cli
```

## Usage

```bash
agent-tasks --help
```
